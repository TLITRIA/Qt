Qt实现爬取网页图片

\n

实现原理

\n

1.获取网页HTML
2.解析HTML得到图片链接
3.下载图片
展示效果图，如下所示：

\n

一、获取到网页的HTML。

\n

这里需要用到Qt网络编程常用的三个类：QNetworkAccessManager、QNetworkRequest和QNetworkReply。
使用上面的类需要引入network模块

\n
QT += network  \n
\n

向网页发送get请求

\n
QNetworkRequest mTestRequest(QUrl(ui->UrlLineEdit->text()));\n//进行网络请求,会返回一个回复，服务器回复结束，QnetworkReply就会释放finished信号\nQNetworkReply *networkReply = mManager->get(mTestRequest);\n//回复结束的时候QnetworkReply会释放readyRead信号\nconnect(networkReply,SIGNAL(readyRead()),this,SLOT(readFinishedSlot()),Qt::UniqueConnection);\n
\n

返回的网页数据会在readFinshedSlot中处理

\n
//sender()函数作用就是得到一个指针，这个指针是：当前槽函数所对应的信号是哪个对象发送的\nQNetworkReply *reply = (QNetworkReply *) sender();\nQByteArray data1 = reply->readAll();    //读取传回来的HTML数据\nQString strData = QString(data1);\nif(strData != "")\n{\n    ui->textEdit->setText("Get website page's data!");\n}\nreply->deleteLater();                   //记得释放内存\ngetPictureLinkFromHTML(strData);        //解析HTML，将图片链接存储在mDownloadList中\ndownloadResource(mDownloadList);        //对这些链接进行下载\n
\n

strData中存放的就是get请求返回的HTML内容

\n

二、解析HTML，获取到图片链接

\n

getPictureLinkFromHTML这个接口通过正则表达式获取到匹配的字符串，并将其处理成可跳转的链接

\n
qDebug()<<"-----------picture------------";\nQStringList qsList;\nqsList.clear();\nQRegularExpression re("<img src=(?<path>\\\\S+?) (?<junk2>.*?)>", QRegularExpression::CaseInsensitiveOption);\n//QRegularExpression globalMatch全部匹配，获取到所有符合这个规则的字符串集合\nQRegularExpressionMatchIterator itor = re.globalMatch(qsContent);\nwhile(itor.hasNext())\n{\n    QRegularExpressionMatch match = itor.next();\n    QString junk2 = match.captured("junk2");\n    QString path = match.captured("path");\n    qsList<<path;\n}\n//处理downloadList数据\nforeach (QString var, qsList) {\n   QString str = var.replace("\\"","");\n   str = str.replace(">","");\n   str = str.replace("'","");\n   qDebug() << "_strData: " << str;\n   mDownloadList << str;\n}\n
\n

QRegularExpression 可以使用正则表达式进行模式匹配 DownloadList中存放的是匹配成功的链接

\n

三、下载图片

\n

先简单说一下，下载图片的原理：仍然是通过get请求获取到链接的HTML数据，但这次我们并不是要对数据做解析，而是将这些数据写入文件中，并将文件重命名为.png格式，这样就实现了通过url下载图片。
OK，上代码

\n
foreach (QString var, _list) {\n       DownloadTipMessage _tips;\n       iImageName++;\n       //可以在resourcename这个属性中加入相对路径，从而将图片下载到指定文件夹\n       _tips.resourcename = QString::number(iImageName);\n       _tips.resourcename += ".png";\n       _tips.urlpath = var;\n       //初始化QFile对象\n       _tips.file = new QFile(_tips.resourcename);\n       //链接地址发送get请求\n       QUrl url(_tips.urlpath);\n       QNetworkRequest request(url);\n       QNetworkReply *currentDownload;\n       currentDownload = mManager->get(request);\n       connect(currentDownload, SIGNAL(finished()),SLOT(downloadFinshined()));\n       connect(currentDownload, SIGNAL(readyRead()),SLOT(downloadReadyRead()));\n       connect(currentDownload,SIGNAL(downloadProgress(qint64,qint64)),this,SLOT(updateProgress(qint64,qint64)));\n       mNetworkmap.insert(currentDownload,_tips);\n    }\n
\n

downloadReadyRead中主要实现将返回的HTML数据写入图片文件中

\n
QNetworkReply *arg= dynamic_cast<QNetworkReply*>(sender());\nif(!mNetworkmap[arg].file->open(QIODevice::WriteOnly)){\n    delete mNetworkmap[arg].file;\n    mNetworkmap[arg].file=NULL;\n    return;\n}\nmNetworkmap[arg].file-><
span class=\"token function\">write(arg->readAll());\n
\n

downloadFinshed中实现文件的写入关闭以及QNetworkReply对象的释放

\n
QNetworkReply *arg= dynamic_cast<QNetworkReply*>(sender());\nmNetworkmap[arg].file->close();\narg->deleteLater();\narg = 0;\n
\n

\n

到这里为止，爬取网页图片到本地的流程已经结束了。
其中第二步，对于html的处理并不是很全面，但是不影响完整的实现流程。（后续，我会再更新对正则表达式的处理）
